# ANT-MOBILE-TARO

[ant-design-mobile-v5](https://github.com/ant-design/ant-design-mobile)`Taro`实现. 同时支持`Vue`和`React`.

📢 注意:

- 如果你只在 h5 端使用, 那还是建议你使用原版. 可以得到官方的支持和及时的版本更新
- 如果你准备在`Taro`的`h5`端和`weapp`端使用的话, 那么可以使用本库. 但是版本内容并不会实时同步原库. 当然欢迎 pr.
